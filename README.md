# How to Start

## Angular2 (using angular-cli)
    $ npm install -g angular-cli
    $ cd angular-2.0
    $ ng serve

## Ionic2 (using ionic)
    $ npm install -g ionic
    $ cd Ionic2-beta11
    $ ionic serve


## vue 1.0 ( using vue-cli )
    
~~~~
$ npm install -g vue-cli
$ cd vue-1.0
$ npm install
$ npm run dev
~~~~

  [vue-CLI](http://vuejs.org/guide/installation.html#CLI)
 

## React v15.3.1 ( bundle with webpack )
